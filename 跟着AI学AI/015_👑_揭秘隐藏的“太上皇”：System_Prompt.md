# 👑 揭秘隐藏的“太上皇”：System Prompt

你们平时跟 ChatGPT、Kimi 聊天的时候，有没有纳闷过这几个问题：
🤔 为什么我什么都没说，它一上来就特别客气，或者自带某种“客服腔”？
🤔 为什么有时候让它扮演个大胆的角色，它却扭扭捏捏说“作为 AI 语言模型，我不能……”？

你以为是你没教好？大错特错！

真相是：在你们俩开始聊天之前，它就已经被“洗脑”了！

这就是 AI 世界里最大的潜规则——【System Prompt（系统提示词）】。

👑 谁才是真正的话事人？
我们平时在聊天框里输入的内容，在技术上叫「User Prompt (用户指令)」。
但在你发送这句指令之前，AI 的开发者早就悄悄给它塞了一段你看不见的指令，这就是「System Prompt (系统指令)」。
如果把 AI 比作一个演员：
- System Prompt 是导演的剧本： 规定了它是谁（人设）、该用什么语气说话（风格）、绝对不能说什么（安全红线）。这是出厂设置，是“思想钢印”。
- User Prompt 是观众的要求： 你让它跳个舞，它会跳，但前提是不能违反导演的剧本。

🕵️‍♂️ “太上皇”的威力有多大
这个隐藏的“太上皇”权重极高！它决定了 AI 的底色。
比如，OpenAI 可能给 ChatGPT 写了这样的系统指令：
“你是一个乐于助人、安全、诚实的 AI 助手。你不能生成暴力、色情内容。遇到你不确定的事实，要承认不知道……”
这就是为什么有时候你怎么“威逼利诱”，它都不肯越雷池一步的原因，因为它骨子里被设定成了“好孩子”。

💡 总结（敲黑板）

普通玩家只能在网页端发 User Prompt；

而高阶玩家（比如程序员接 API），是可以自己改写 System Prompt 的！目前各个平台或者风转的网页端也支持设置系统提示词。

掌握了它，你才能真正定制出专属于你的独特 AI！
